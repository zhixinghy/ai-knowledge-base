# Cortex 快速上手

Cortex 是一个基于 RAG + Agent 的企业知识库智能助手:把你的 PDF 文档上传进来,它就能基于这些资料用中文回答问题,并标注答案出处。本文带你三步上手。

## 一、注册 / 登录

- **匿名试用**:不注册也能先体验。问答和上传合计有几次免费额度(默认 5 次,具体以页面提示为准),用完后会弹出登录窗口。
- **注册账号**:点击首页右上角的「登录 / 注册」,或在试用额度用尽时弹出的窗口里切换到「注册」。填写用户名和密码即可,无需邮箱或手机号。
- **登录的好处**:登录后不限使用次数,并且拥有一个**专属的私有知识库**——你上传的文档只有你自己能看到和检索。

## 二、上传文档,建立你的知识库

1. 进入左侧导航的「**知识库**」页面。
2. 把 PDF 文件拖到上传区域,或点击选择文件。
3. 上传后系统会自动处理:**解析 → 切块 → 向量化 → 入库**,文档会先显示「解析中」,处理完成后变为「就绪」。就绪后即可在问答里检索到。

> 说明:目前**仅支持 PDF 格式**。建议上传带有可选中文字的 PDF;纯扫描件 / 图片型 PDF 可能无法提取文本。

## 三、开始提问

1. 进入「**问答**」页面。
2. 选择一个对话模式(见下),在底部输入框输入问题并发送。
3. 在「文档助手」模式下,系统会先从你的知识库检索相关内容,再据此作答,并在回答下方给出**出处**(来自哪篇文档、第几页)。

## 三种对话模式(速览)

| 模式 | 用途 |
| --- | --- |
| **文档助手** | 优先依据你上传的文档作答,并附出处 |
| **客服** | 基于公共的客服资料(FAQ / 政策等)友好解答 |
| **工具增强** | 可调用计算器、联网搜索等工具完成复合任务 |

> 各模式的详细区别见《Cortex 功能与对话模式》。

## 小提示

- **切换模式会清空当前对话**——因为换模式等于换了场景和知识范围,属正常设计。
- 回答下方的「出处」可以点开,核对答案依据的原文片段和页码。
- AI 生成的内容可能有误,重要信息请以原文或官方资料为准。
